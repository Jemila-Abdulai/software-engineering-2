### Pull Requests

To commit code, we will operate a branch and PR system. 

 - Create a branch from `master`
 - Make your changes in a branch
 - Pull from `master` before committing
 - Create a PR
 - Seek someone to review

### In your Pull Request

Include:
 - Issue number
 - Summary of changes
 - How tested (in brief)
